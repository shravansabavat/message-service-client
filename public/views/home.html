A simple application which is developed in angular.js running on node server to save messages posted by a user of this application.
This application calls a message-service micro service (implemented in node.js) - which offers
  • GET (list messages) - List existing messages
  • GET (details of a message) - get details of the message.
  • Delete - delete a message.
  • POST - save a message.

Features

  • Click on the Messages tab opens a new form to input a string
  • On form submission, sends the string to the micro service, stores into a list of messages and updates the list on the Messages page
  • Click any message in the list of messages to see the details
  • Details include message text, date it is posted, and displays if it is a palindrome
  • Click on 'X' to delete a message from the list.
