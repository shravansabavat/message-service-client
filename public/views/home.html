A simple application which is developed in angular.js running on node server to find out if a given string is palindrome or not.
This application calls a palindrome-service node.js micro service - which offers
  • GET (list palindromes) - List existing palindromes
  • GET (details of a palindrome) - get details of the palindrome.
  • Delete - delete a palindrome.
  • POST - Check if a given string is palindrome, if yes then store it.

Features

  • Click on the Check palindrome tab opens a new form to input a string
  • On Check palindrome form submission, validates the string on the micro service, stores (if valid) into a valid list of palindromes and updates the list on the Check palindrome page
