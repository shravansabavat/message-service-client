A simple application which is developed in angular.js running on node server to find out if a given string is pallyndrome or not.
This application calls a pallyndrome-service node.js micro service - which offers
  • 1)GET (list pallyndromes) - List existing pallyndromes
  • 2)GET (details of a pallyndrome) - get details of the pallyndrome.
  • 3)POST - Check if a given string is pallyndrome, if yes then store it.

Features

  • Click on the Check Pallyndrome tab opens a new form to input a string
  • On Check Pallyndrome form submission, validates the string on the micro service, stores (if valid) into a valid list of pallyndromes and updates the list on the Check Pallyndrome page
